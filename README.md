# Agro AI Disease Detection

AI-based crop disease detection system for Sri Lankan agriculture.

This project detects **Rice Leaf Blast** from leaf images using a CNN trained with TensorFlow/Keras.  
The model is trained on a **clean, deduplicated dataset** and includes a **CLI inference tool**.
